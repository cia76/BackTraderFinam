# BackTraderFinam
Часть [Системы "Финансовая Лаборатория"](https://github.com/cia76/FinLabPy) от Игоря Чечета

### Для чего нужен
Чтобы торговые системы, написанные для BackTrader, можно было поставить на автоматическую торговлю с брокером [Финам](https://www.finam.ru/).

### Установка провайдера
1. По инструкции установите и настройте [Систему "Финансовая Лаборатория"](https://github.com/cia76/FinLabPy)
2. Для проверки ее работоспособности запустите примеры из папки **Brokers/Examples**
3. Установите [BackTrader](https://backtrader.com/) (pip install backtrader)
4. Для проверки ее работоспособности запустите примеры из папки **Backtrader/Examples**
